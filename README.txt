README:
1) Please install node dependencies using npm install for both client and server code.
2) Download the server code from the master branch.(A blank for client code will be downloaded. Follow next step.)
3) Download the client code from the client branch into the photo-crop-app folder.
4) Run the Node application from the src directory by running the command node index.js. 
5) The database is initialized by going to localhost:port/events route. The port number could be different depending on the local machine running the application.
6) The db.json file currently contains the database. If required to initialize follow step 2.
7) Run the angular server from the photo crop folder using ng serve. 
8) Go to localhost:port/. The port number depending on the on the local machine running the application.
9) The data will be displayed to the user.
