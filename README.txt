README:

1) Run the Node application from the src directory by running the command node index.js. Download the server code from the master branch.
2) The database is initialized by going to localhost:port/events route. The port number could be different depending on the local machine running the application.
3) The db.json file currently contains the database. If required to initialize follow step 2.
4) Run the angular server from the photo crop folder using ng serve. Download the client code from the client branch into the photo-crop-app folder.
5) Go to localhost:port/. The port number depending on the on the local machine running the application.
6) The data will be displayed to the user.
